About Me

Hey everyone! My name is Brent, and I'm new to the web development world. I'm excited to get my start in this new chapter, bringing my experience from my previous "lives".

I have come from a wide array of backgrounds to join this part of the workforce. Notably, I've been an operations and logistics coordinator for a robotics company, a mortgage broker, and a ticket broker, all in a span of 20 years. While each of these jobs were pretty different, they have all taught me lessons that I look forward to bringing to the table here.

I also have a passion for working in my kitchen, cooking up a wide variety of dishes, from a wide variety of cuisines, for my family to enjoy. I look forward to bringing the same energy and creativity I've used in my musical and culinary life to this project as well.

I'm excited to join a team of coders where we can collaborate on projects, bounce ideas off each other and learn from each other's experiences. Working together as a team, there's no end to what we can bring to the table!

Brewery Tour Creater

For my API hack project, I decided to create an app that would search a user-defined area for breweries, brewpubs, bottling houses and the like. The user can then select one of the locations, and the app will reorganize the list based on the proximity to the starting point.

While the page is constructed using HTML and CSS, the main functionality is in the Javascript and JQuery, including a function to calculate the distance between points based on the latitude and longitude of the locations.

You can access the code through my GitHub account here.
